# Kickoff GatsbyJs

A starting point using ReactJs, bundled with awesome GatsbyJs.

## Stack

* [GatsbyJs](https://www.gatsbyjs.org/)
* [React.js](https://reactjs.org/)

## To use

* Fork or download this repository
* Ready!

## Setup

Run:

```
yarn install
```

## Development

To start development server

```
yarn start
```

## Deployment

```
yarn run build
yarn start
```
